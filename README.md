# RuleBot - Rule-Based AI Chatbot 🤖

This is my first project completed during the DecodeLabs AI Internship.

RuleBot is a simple rule-based chatbot built using Python. It responds to predefined user inputs and provides basic information about AI, Machine Learning, and Deep Learning.

## Features

* Greeting responses
* AI-related questions
* Menu-based interaction
* Exit command

## Technologies Used

* Python 3

## How to Run

```bash
python task1.py
```

## Sample Commands

* hello
* hi
* how are you
* what is your name
* what is ai
* what is machine learning
* what is deep learning
* exit
